# 🌐 Personal Portfolio Website

A responsive and modern portfolio website built using **HTML, CSS, and Bootstrap 5**. This project showcases personal information, technical skills, projects, achievements, and contact details in a clean and professional layout.

---

## 🚀 Features

✅ Responsive Navigation Bar

✅ Hero Section with Profile Picture

✅ About Me Section

✅ Skills with Progress Bars

✅ Projects Showcase

✅ Achievements Section

✅ Contact Form

✅ Mobile-Friendly Design

✅ Clean and Modern UI

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- Bootstrap 5

## 🔧 Installation

1. Clone the repository

```bash
git clone https://github.com/sriharsha0x1/Personal-Portfolio-Website.git
```

2. Navigate to the project folder

```bash
cd Personal-Portfolio-Website
```

3. Open `index.html` in your browser.
